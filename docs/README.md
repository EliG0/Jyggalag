## 🚀 Запуск приложения

1. Перейдите в раздел **Releases** и скачайте актуальную версию.
2. Распакуйте в любую удобную папку.
3. Запустите `jyggalag.exe`.

# 🔍Troubleshooting

* Ошибка "This application requires a Java Runtime Environment":
  Исполняемый EXE-файл не может обнаружить Java в системном пути PATH. Установите JDK 26 или укажите корректный путь в
  настройках среды Windows.
